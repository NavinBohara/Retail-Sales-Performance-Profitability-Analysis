# 03 - Power BI Dashboard

This folder contains visualization deliverables for business intelligence reporting.

## Contents
- **Retail_Sales_Dashboard.pbix**
  - Fully interactive Power BI dashboard with KPI insights
- **DAX_Measures.txt**
  - Complete list of DAX measures used for calculations
- **Dashboard_Screenshots/**
  - High-resolution screenshots for documentation and GitHub preview

## Key Insights Visualized
- Total Sales, Total Profit, and Profit Margin KPIs
- Category and region performance benchmarks
- YoY and seasonal sales analysis
- Discount impact on profitability
