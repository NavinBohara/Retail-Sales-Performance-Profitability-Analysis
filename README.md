# 🛍️ Retail Sales Performance & Profitability Analytics

An end-to-end data analytics project focused on assessing retail sales performance across India.  
Developed using **Excel for data cleaning**, **MySQL for analytical modeling**, and **Power BI for business intelligence & visualization**.

## 📌 Project Overview

Retail business data was analyzed to measure revenue performance, profitability trends, and discount strategy effectiveness across multiple states, regions, and product categories in India.

The project delivers actionable insights to improve operational efficiency and maximize profit growth.

---

## 🎯 Objectives

- Clean and standardize raw retail transactional data
- Analyze category, region, and city-level performance
- Identify high-profit and low-performing market segments
- Evaluate discounting strategies and their impact on profitability
- Present findings using a modern BI dashboard
