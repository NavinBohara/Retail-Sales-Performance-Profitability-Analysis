# 02 - SQL Scripts

This folder contains SQL scripts used for data modeling and business analytics.

## Files Included
- **Create_Tables.sql**
  - SQL statements for creating the database schema.
  - Ensures structural integrity and consistent datatypes.

- **Retail_Sales_Analysis_Queries.sql**
  - KPI calculations and analytical queries including:
    - Category-wise revenue and margin analysis
    - Region and city-level profitability
    - YoY and seasonal sales trend analysis
    - Discount performance segmentation

## Database System
- MySQL (Recommended version: 8.0 or later)
